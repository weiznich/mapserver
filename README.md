# Simple OpenStreetMap-Mapserver

This is a simple OSM-Mapserver written in Ruby on Rails.


## Install

Clone the Git-repository and run the following commands:

    $ bundle install
    $ cabal install aeson
    $ make

Then you can run a server via

    $ rails server

## Systemrequirements
Known to work with:

  ghc(7.6.2)
  ruby(1.9.3)
  rails(3.2.12)
  gem(1.8.24)
  bundle(1.2.3)
  cabal-install(0.14.0)
